# Documentation Index

This repository uses a document-centric layout with deterministic harness scripts.

Folders and files:

- `docs/HERMES_TELEGRAM_INSTALL_PLAN.md` — installation plan, environment state, and low-tier autorunner.
- `docs/copilot-instructions.md` — deterministic Copilot usage rules and workflow.
- `docs/N8N.md` — n8n integration and validation notes.
- `skills/README.md` — lightweight skill reference for local script execution.
- `skills/docs/COPILOT_SKILL.md` — Copilot-specific skill documentation.

Use the `skills/scripts/` tools to update the repository state and documentation instead of editing generated files directly.
